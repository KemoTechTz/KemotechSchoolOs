# Abela Pre & Primary School — Frontend Version 3

A premium multi-page static website for **Abela Pre & Primary School**, located in **Mkuranga, Pwani, Tanzania**. Built with only:

- HTML
- Tailwind CDN
- Vanilla JavaScript
- Custom CSS

No backend is connected yet. Forms are backend-ready and use clear `name` fields plus `data-endpoint` attributes where needed.

## Pages included

- `index.html` — premium homepage
- `about.html` — school story, mission, vision and values
- `academics.html` — Tanzania curriculum-aligned academic pathway
- `admissions.html` — admission process, visit section, documents and fees preview
- `apply.html` — multi-step admission application portal demo
- `portal.html` — parent portal dashboard demo using localStorage
- `fees.html` — sample fees structure
- `school-life.html` — clubs, meals, sports, transport and reading culture
- `teachers.html` — teacher/staff profile layout
- `gallery.html` — filterable school gallery
- `photo-library.html` — new V3 image source library, licensing notes and photoshoot shot list
- `news.html` — announcements/news page
- `contact.html` — backend-ready enquiry form

## Image usage warning

Do not copy photos directly from Feza, Aga Khan, IST, Braeburn, or any real school website unless written permission is granted. These are included as visual inspiration references only. For production, replace remote placeholder photos with real Abela photos or properly licensed stock images.

## How to run

Open `index.html` directly in a browser, or serve the folder with a local static server.

Example:

```bash
python -m http.server 5500
```

Then visit:

```txt
http://localhost:5500
```
