# Abela School Image Strategy

Version 3 uses remote placeholder images for design preview only. Before launch, replace them with real Abela school photos inside this folder.

Recommended folders:

- `hero/` — homepage, admissions, academics hero photos
- `classroom/` — learning, reading, writing, teacher support
- `campus/` — gate, buildings, playground, assembly area
- `staff/` — head teacher, class teachers, administration
- `school-life/` — clubs, meals, sports, transport, activities
- `photo-library/` — curated shortlisted assets and license notes

Never copy real photos from Feza, Aga Khan, IST, Braeburn, or any school website without written permission. Use those schools as visual inspiration only.
